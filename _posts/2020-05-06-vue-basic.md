## Vue 기본 기능    

#### 인스턴스 생성
뷰로 개발할 때 필수로 생성해야 하는 코드

```
new Vue();
```

참고로 뷰 인스턴스는 변수에 담아서 확인해 볼 수 있다.
```
const app = new Vue();
console.log(app);
```
